# Shopping Cart Spring Boot Project

This is a simple **online shopping cart** application built using **Spring Boot (MVC)**, **Thymeleaf**, and **MySQL**.

## 🚀 Features
- Product listing
- User authentication (admin & customer)
- Shopping cart management
- Order processing
- Email notifications (optional)

## 💻 Technologies Used
- Java 17
- Spring Boot 3.2.3
- Spring Data JPA
- Spring Security
- Thymeleaf
- MySQL
- Maven
